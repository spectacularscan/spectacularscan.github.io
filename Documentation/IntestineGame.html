update()

This function first checks if the sleepTimer is greater than 0 and if it is, it reduces the value by 0.1 seconds. This is done to make sure the game movement mechanics feel smoother. Then it checks which key was pressed and based on which one it calls another method that corresponds to it. Thus, allowing the user to move left, right, up or down.
